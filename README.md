# Amazon Clone (Landing Page)

A responsive Amazon-inspired landing page built using HTML5 and CSS3.  
This project focuses on recreating the layout and UI structure of Amazon’s homepage while practicing modern frontend development, clean design, and responsive styling.

---

## Features

- Amazon-style homepage layout  
- Responsive navigation bar  
- Search bar with category dropdown  
- Hero banner section  
- Product category cards  
- Multiple shopping sections with images  
- Multi-column footer  
- Fully responsive design for different screen sizes  

---

## Tech Stack

- **HTML5** – Structure and semantic markup  
- **CSS3** – Styling, layout, and responsiveness  

---

## Note

This is a **frontend-only educational project**.  
It does not include:
- Backend services  
- User authentication  
- Shopping cart functionality  
- Payment integration
