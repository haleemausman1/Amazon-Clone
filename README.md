# Amazon Clone (Landing Page)

A responsive Amazon-inspired landing page developed using **HTML5** and **CSS3**. This project recreates the look and feel of Amazon's homepage while focusing on clean layout design, responsive styling, and modern frontend development practices.

## Features

* Amazon-style homepage layout
* Responsive navigation bar
* Search bar with category selection
* Hero banner section
* Product category cards
* Shopping sections with images
* Multi-column footer
* Responsive design for different screen sizes

## Tech Stack

* **HTML5** – Structure and semantic markup
* **CSS3** – Styling, layout, and responsive design

> **Note:** This is a frontend-only project. It does not include JavaScript functionality, backend services, authentication, shopping cart, or payment integration.

## Installation

Clone the repository:

```bash
git clone https://github.com/haleemausman1/Amazon-Clone.git
```

Open the project folder and launch **index.html** in your preferred web browser.
